# Manual de Usuario - Módulo de Hospitalización

Este manual proporciona una guía paso a paso para utilizar el módulo de Hospitalización, diseñado para facilitar el seguimiento de pacientes, la gestión de camas y la coordinación de tareas médicas.

## 1. Introducción al Panel de Control
El panel de control es la pantalla principal donde podrá visualizar el estado de todas las habitaciones y camas de la clínica en tiempo real.

### Elementos Principales:
*   **Tarjetas de Habitación:** Cada recuadro representa una habitación física.
*   **Indicadores de Cama:** Dentro de cada habitación, verá iconos que representan las camas.
*   **Estadísticas Generales:** En la parte superior, verá un resumen de ocupación, tareas pendientes y alertas.

## 2. Cómo leer el Estado de las Camas
Los colores y símbolos le indican rápidamente qué está sucediendo:

*   **Verde:** Cama disponible (libre).
*   **Azul/Cian:** Cama ocupada por un paciente.
*   **Amarillo:** Cama reservada o con alta programada próximamente.
*   **Icono de Hombre/Mujer:** Indica el sexo del paciente en la cama.
*   **Burbuja Roja con Número:** Indica cuántas tareas médicas están pendientes para ese paciente.
*   **Triángulo de Advertencia:** Indica que hay tareas vencidas o que el paciente no tiene un diagnóstico registrado (CIE10).

## 3. Gestión de Pacientes (El "Cajón" de Detalles)
Al hacer clic en una cama ocupada (excepto si tiene rol de Secretaría), se abrirá un panel lateral derecho con la información detallada.

### Pestañas Disponibles:
1.  **Detalles de Atención:** Aquí se registra la evolución médica diaria. Puede ver una línea de tiempo de las evaluaciones pasadas.
2.  **Tareas:** Lista de actividades pendientes (ej: "Administrar medicación", "Curación").
    *   **Verde:** Tarea completada.
    *   **Naranja:** Tarea por vencer.
    *   **Rojo:** Tarea vencida.
3.  **Reservas:** Información sobre si la cama está separada para un próximo ingreso.

## 4. Funciones Avanzadas
### Sincronización (Solo Administradores/Coordinadores)
Si nota que falta un paciente que acaba de ingresar por Admisión, utilice el botón **"Sincronizar"** en la parte superior derecha. Esto traerá los datos actualizados del sistema central (Sisclin).

### Ordenamiento Inteligente
Haga clic en el botón de la "varita mágica" (**Ordenamiento Inteligente**) para organizar las habitaciones de modo que las que tienen más pacientes o tareas críticas aparezcan primero.

### Filtros
Utilice la sección de filtros para buscar una habitación específica o ver solo las camas que están libres o reservadas.

## 5. Vista para Televisor (Modo TV)
Si desea proyectar el estado de hospitalización en una pantalla grande en el área de enfermería, haga clic en el icono de **"Pantalla"** (Vista TV). Esta vista es de solo lectura y se actualiza automáticamente cada pocos segundos.
